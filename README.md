# Assignment-2

This project contains the Lab 5 exercises for Operating Systems,It includes examples for process creation, linking, and loader behavior.
how to compile 
Run the following command in the terminal:
./ex1_fork
./output_program
./simple_program

purpose of Each Program
ex1_fork.c: Tests process creation using fork().
file1.c + file2.c: Used together to show how linking works in GCC.
simple_program.c: Basic program to check dynamic loader output using ldd.
